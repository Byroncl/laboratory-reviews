🔌 CONEXIONES:

App Init:
  app.component.ts → ngOnInit()
    ↓
  dispatch(loadAuthFromStorage)
    ↓
  AuthReducer → Restaura token & user desde localStorage
    ↓
  AppComponent usa Router

Routes:
  /auth/login     → LoginComponent (public)
  /auth/register  → RegisterComponent (public)
  /dashboard      → DashboardComponent (protected por authGuard)
  /               → Redirige a /auth/login
  /**             → Redirige a /auth/login

Services:
  AuthService → HTTP calls a /api/auth/login y /api/auth/register

Store:
  AppComponent dispatch → AuthActions
    ↓
  AuthReducer → Actualiza state (user, token, isAuthenticated)
    ↓
  AuthEffects → Escucha actions y hace side effects (API calls, navigate, localStorage)
    ↓
  Selectors → Componentes leen state (selectUser, selectIsAuthenticated)

Interceptors:
  JWT Interceptor → Agrega token a requests
  Error Interceptor → Maneja errores HTTP
