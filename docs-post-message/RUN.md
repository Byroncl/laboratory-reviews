# 🚀 Cómo Ejecutar

Es muy simple. Solo ejecuta estos dos comandos:

```bash
npm install
npm start
```

Eso es todo. La documentación se abrirá en **http://localhost:3000** 📖

---

## En Command Prompt

```bash
cd "C:\Users\byron\Documents\laboratory-reviews\docs-post-message"
npm install
npm start
```

## En PowerShell

```powershell
cd "C:\Users\byron\Documents\laboratory-reviews\docs-post-message"
npm install
npm start
```

## En Terminal (macOS/Linux)

```bash
cd path/to/docs-post-message
npm install
npm start
```

---

## Eso es todo ✅

- ✅ npm install — instala dependencias
- ✅ npm start — inicia servidor de desarrollo
- 📖 Se abre automáticamente en http://localhost:3000

Presiona **Ctrl+C** para detener el servidor.

---

## Otros Comandos

```bash
npm run build    # Build para producción
npm run serve    # Sirve el build
npm run clear    # Limpia cache
```
