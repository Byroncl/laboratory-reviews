# ✅ INICIO AQUÍ

## 🚀 En 2 Pasos

```bash
npm install
npm start
```

**Eso es todo.** La documentación abrirá en http://localhost:3000

---

## 📝 Versión Larga

### Paso 1: Abre Command Prompt o PowerShell

En Windows:
- Presiona `Win + R`
- Escribe `cmd` o `powershell`
- Presiona Enter

### Paso 2: Ve a la carpeta

```bash
cd "C:\Users\byron\Documents\laboratory-reviews\docs-post-message"
```

### Paso 3: Instala dependencias (primera vez)

```bash
npm install
```

Espera a que termine (1-2 minutos).

### Paso 4: Inicia el servidor

```bash
npm start
```

Debería ver:

```
✅ Docusaurus website is running at: http://localhost:3000/
```

### Paso 5: Abre el navegador

Visita: **http://localhost:3000**

---

## 🛑 Para Detener

Presiona **Ctrl+C** en la terminal.
