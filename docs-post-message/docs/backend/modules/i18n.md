# Módulo I18n 🌍

Maneja las traducciones de la aplicación para inglés y español.

## Descripción General

Provee endpoints para obtener las traducciones usadas por el frontend.

## Arquitectura

⚠️ **Existen dos sistemas i18n paralelos** (problema a resolver):

1. **TranslationService** (Singleton) — Usado por guards/filtros
2. **I18nService** (Scoped por petición) — Usado por módulos

Ver [Problema de Inconsistencia I18n](../issues/i18n-inconsistency.md)


## Detección de Idioma

El idioma se detecta desde el header `Accept-Language`:

```
Accept-Language: es-MX,es;q=0.9,en;q=0.8
→ 'es'

Accept-Language: en-US,en;q=0.9
→ 'en'
```

## Endpoints

| Endpoint | Método | Propósito |
|----------|--------|---------|
| `/i18n/translations/:language` | GET | Obtener traducciones para el idioma |

## Uso en Frontend (Angular)

```typescript
// Obtener traducciones
this.http.get('/i18n/translations/es').subscribe(translations => {
  // Usar traducciones
  console.log(translations.auth.required);  // 'Autenticación requerida'
});
```
