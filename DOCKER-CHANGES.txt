🔍 TROUBLESHOOTING:

Si el backend no responde:
  1. Ver logs: docker-compose logs nestjs-app
  2. Verificar compilación: docker-compose build --no-cache
  3. Verificar puerto: docker exec mi_nestjs netstat -tln
  4. Reiniciar: docker-compose restart nestjs-app

Si MongoDB no conecta:
  1. Ver logs: docker-compose logs mongodb
  2. Ping desde backend: docker exec mi_nestjs ping mongodb
  3. Verificar .env: echo $MONGODB_URI

Si MinIO no funciona:
  1. Ver logs: docker-compose logs minio
  2. Health check: docker exec mi_minio curl http://localhost:9000/minio/health/live
